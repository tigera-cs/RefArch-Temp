# Calico Unified Platform — interactive reference pages

A small set of interactive web pages that present the **Calico Unified Platform** to
prospects and internal teams. Each page is a single **self-contained HTML file**
(inline CSS + JS, images embedded, no build step, no dependencies) hosted on GitHub
Pages. They share one consistent top navigation and are tuned for desktop.

**Start here:** https://tigera-cs.github.io/RefArch-Temp/

## Pages

| Page | What it is |
|------|------------|
| [Home](https://tigera-cs.github.io/RefArch-Temp/) | Launcher page with three tiles into the experiences below. |
| [Architecture](https://tigera-cs.github.io/RefArch-Temp/Calico_Architecture_v1.html) | "How Calico sits in your cluster" — an interactive component diagram with OSS / Enterprise / Cloud editions and step-through "how it works" workflows. |
| [Envision](https://tigera-cs.github.io/RefArch-Temp/Calico_Envision_v1.html) | A short wizard that assembles a reference architecture tailored to your industry and priorities. |
| [Unified Platform](https://tigera-cs.github.io/RefArch-Temp/Calico_Unified_Platform_v1.html) | The platform at a glance — Networking, Security, Observability and VM Migration layers you drill into. |
